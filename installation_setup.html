Installation & Setup

Installing Civic Builder

Web Application

To use Civic Builder via the web application, follow these simple steps:

  1. Open your preferred web browser.
  2. Visit the Civic Builder website at [insert web address].
  3. Create a new account if you haven't already, or log in to your existing account.
  4. Start using Civic Builder directly from your browser. No installation is required.

Standalone Offline Version (Windows Only)

If you prefer an offline version of Civic Builder on Windows, follow these steps:

  1. Download the Civic Builder installer from the official website.
  2. Double-click the downloaded installer file to begin installation.
  3. Follow the on-screen instructions to complete the installation process.
  4. Once installed, launch Civic Builder from your desktop shortcut or Start menu.

System Requirements

Ensure your system meets the following minimum requirements for optimal performance:

  • Operating System: Windows 10 or later for the standalone version. The web application is compatible with any modern OS with a supported browser.
  • Processor: Multi-core processor (Intel i5 or equivalent recommended).
  • RAM: 8GB or more recommended.
  • Graphics: Dedicated GPU recommended for smooth rendering and performance.
  • Storage: Sufficient disk space for installation and project files.

Setting Up Civic Builder

Web Application

The web application setup involves:

  1. Logging in to your account on the Civic Builder website.
  2. Accessing project files and saving options within the application.
  3. Exploring available features and tools through intuitive menus and user guides.

Standalone Offline Version

After installing the standalone version, set up Civic Builder by:

  1. Launching the application from the desktop shortcut or Start menu.
  2. Configuring initial settings such as preferred language, default project folder, and user profile.
  3. Importing required imagery and GIS data for detailed cityscape modeling.
  4. Exploring advanced features for precise model generation and export options.

With Civic Builder installed and set up correctly, you're ready to start creating detailed 3D cityscapes with ease. Refer to the comprehensive documentation for additional guidance and tips.
